# PROJECT STRUCTURE

```bash
stitches_backend/
├── app/
│   ├── __init__.py          ← creates the Flask app (app factory)
│   ├── models/
│   │   ├── __init__.py
│   │   ├── user.py          ← User model
│   │   ├── product.py       ← Product, Yarn models
│   │   └── order.py         ← Order, OrderItem models
│   ├── routes/
│   │   ├── __init__.py
│   │   ├── auth.py          ← /register, /login, /logout
│   │   ├── products.py      ← /products GET, POST, PUT, DELETE
│   │   └── orders.py        ← /orders GET, POST
│   ├── schemas/             ← serialization (marshmallow or manual)
│   │   ├── user_schema.py
│   │   └── product_schema.py
│   └── utils/
│       ├── auth_helpers.py  ← JWT decode, role checks
│       └── decorators.py    ← @admin_required, @login_required
├── migrations/              ← auto-generated by Flask-Migrate
├── config.py                ← Dev, Test, Prod config classes
├── seed.py                  ← seed your DB with sample data
├── run.py                   ← entry point (replaces bare app.py)
├── requirements.txt
└── .env                     ← SECRET_KEY, DATABASE_URL etc.


```
